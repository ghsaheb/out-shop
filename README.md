# OutShop

OutShop is a checkout system for adding products in shopping cart

### Prerequisites

sqlite-jdbc-3.8.11.2.jar

## Getting Started
1. Download and place sqlite-jdbc-3.8.11.2.jar into lib folder.
2. Then Run the project using run.sh

```
sh run.sh
```
